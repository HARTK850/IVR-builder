ברוך הבא ל-Yemot AI Studio (Pro Edition)

המערכת החכמה בעולם לפיתוח שלוחות וניהול ימות המשיח.
בגרסת הפרו הוספנו תמיכה מלאה בהפרדה בין שלוחות רגילות לבין שלוחות API המנוהלות מול GitHub ופורסות אוטומטית ל-Vercel.

בונה שלוחות (רגיל)

ממשק ייעודי ליצירה והגדרה של שלוחות רגילות בימות המשיח (הגדרות ext.ini).

  • הגדרת השמעת קבצים, תפריטים וניתובים.
  • הפעלת פקודות API לימות אוטומטית לאחר אישור.
  • שואב ידע אדיר מקובץ knowledge_base.txt.

מפתח API (GitHub & Vercel)

מושלם למפתחי Node.js. כותב קוד לשלוחת API ופורס אוטומטית לשרת.

  • יצירת פונקציונליות מורכבת (Express.js).
  • העלאה ישירה ל-GitHub דרך API.
  • הגדרת הניתוב בימות לכתובת ה-Vercel שלך.
